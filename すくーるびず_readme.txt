                              "すくーるびず"

あらすじ：
ある年の初夏、例年を大きく上回る気温と湿度に見舞われる異常気象が続いていた。
とある学校の生徒、小夏はこのあまりの暑さに耐えきれず、学校へ半袖ミニスカートの夏服で登校するに決める。
しかし、この学校では制服移行期間は未だ実施されておらず、校則に違反していることを生徒会に知られてしまうとすごい罰則をくらうらしい。
そんなことも忘れて夏服で登校してしまった小夏の前に生徒会長、冬香が立ちはだかる。

　　　　　　　"すくーる"　での　"くーるびず"　をかけた戦い

意地でも半袖ミニスカが着たい小夏と、校則絶対遵守の冬香の”暑い”シューティングバトルが今始まる

------------------------------------------------------

遊び方説明



矢印キー：8方向移動

Shiftキー：押しながら矢印キーで低速移動

Ctrl または Zキー：
①各種決定キー

②うきわを装着する　
　うきわを装着した状態で敵の弾に当たることで敵の弾を反射することができる
　5秒経過するとストックが1回復し、3つまでためることができる（←膨らますのには時間がかかりますからねぇ
　ストック1つにつき8秒間うきわを装着することができる
　一度に複数同時に使用することもできるが、ダウンしてしまうと装着しているうきわはリセットされてしまう

Escキー または Pキー：ポーズメニュー

Rキー：プレイ中の難易度を最初からやり直す



難易度は　easy,normal,hard の3種類から選べる

残機：4つ(スタート時＋かき氷3つ分）

敵の弾か敵本体に自機が衝突すると残機を1つ失う
残機が0になるとゲームオーバー
コンテニューはなし

敵のHPは敵の後ろの丸の色で判断でき、緑(満タン時)→赤(HP0)と変化する
敵の行動パターンは敵のHPに依存して変化する


弾の種類

青：敵が撃ってくるこちらがうきわで反射できる弾
黒：敵が撃ってくるこちらがうきわで反射できない弾
赤：敵の弾をうきわで反射した後の弾
　　反射後は敵を少しホーミングしながら加速する




華麗に弾幕をよけて、うきわで反撃！
生徒会長を倒して夏服を奪還せよ！！

製作後記

本作品は、プログラマ　ru としての初めてのゲーム作品であり、始めはやはりシューティングゲームからかなと思い製作を始めました。
今までにはない斬新なもの目指した結果、本作品のようにシューティングゲームなのにこちらがシューティングしないという発想に至りました。
現実世界でも猛暑は続いており、年々進む地球温暖化の中で人々がいかに快適に過ごすかを考えているときに、そもそも学校生活に制服移行期間なんか必要ないんじゃないか、各自が気温に合わせて制服を変えればいいじゃないかという思いを抱き始め、本作品のタイトルにもなっている、学校（スクール）と、衣服の軽量化キャンペーン（クールビズ）を掛け合わせた造語、”すくーるびず”を思いつきました。（特許取りたいくらい）
プレイヤーに爽快感を味わってもらうべく、夏、海、女の子をテーマにBGMも南国調なさわやかな感じで仕上げてみました。
まだまだUIやプログラム、コンテンツなど至らない点はございますが、楽しんでいただけたら幸いです。
